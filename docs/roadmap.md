# Roadmap

## Prototype today
- shared schemas
- sim-core CLI
- result JSON
- beta-style local dev-viewer browser with curated launcher

## Beta slice now
- 2 tanks
- 4 prototype shells across AP and HE
- 11 curated showcase scenarios with matching result/debug outputs
- built-in scenario browser and launcher over the curated content pack
- featured quick-access strip and reset-to-default launcher flow
- replay controls
- debugable event log
- compact debug inspector over curated debug JSON
- basic x-ray inspection
- manifest-driven regression refresh command for the curated beta pack

## Before wider beta
- search/filtering or favorite presets in the scenario browser
- one curated miss / overshoot showcase
- a third tank or variant set
- more historically grounded shell/tank data
- better bundle splitting for the dev viewer

## Later
- better damage visuals
- richer module logic
- maybe Rust/WASM core
